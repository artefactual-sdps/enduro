Help & Support

Get the assistance you need to make the most of Enduro

Navigate to the "Ingest Content" section from the home page or use the main menu. You can either drag and drop files directly or browse to select them. The system will automatically validate and process your uploads according to configured workflows.

Go to the "Monitor Progress" section to view real-time status updates for all your transfers. You can filter by status, date range, or search for specific items. Each transfer includes detailed logs and any alerts that need attention.

Enduro supports a wide range of file formats including documents (PDF, DOCX), images (JPEG, PNG, TIFF), videos (MP4, AVI, MOV), and many others. The system automatically identifies and validates formats during the ingest process. Contact support for specific format requirements.

Navigate to "Manage Settings" and select "Storage Locations". Here you can add, edit, or remove storage destinations. Each location can be configured with specific policies, retention rules, and access controls. Consult the documentation for detailed configuration options.
